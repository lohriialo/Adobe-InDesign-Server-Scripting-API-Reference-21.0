Point (SUI)

Defines the location of a window or UI element. Contains a 2-element array.

Specifies the origin point of an element as horizontal and vertical pixel offsets from the origin of the element's coordinate space. A Point object is created when you set an elementâ€™s location property. You can set the property using a JavaScript object with properties named x and y, or an array with 2 values in the order [x, y].
